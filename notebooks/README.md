# Notebooks oficiales MaxVSC-SW

Este directorio contiene los notebooks de demostración del simulador RMS VSC-HVDC:

- `01_introduccion_simulador.ipynb`
- `02_modo_PQ.ipynb`
- `03_modo_VdcQ.ipynb`
- `04_integradores_dt_adaptativo.ipynb`

Los notebooks usan únicamente la API pública:

- `vscsim.api.simulation.run_simulation`
- `vscsim.utils.exporter.*`
- `vscsim.api.batch.run_batch` (en ejemplos avanzados)
